# mobile-invitation-front
chatGPT로 만드는 모바일 청첩장 프론트입니다.

# Commit Rules
```
NEW: 신규 파일 및 기능 추가
IMP: 기능 개선
SCH: 설정 변경
CLN: 코드 정리 및 리팩토링
BUG: 버그 픽스
TST: 테스트
DOC: 문서작업
RMV: 삭제
```
